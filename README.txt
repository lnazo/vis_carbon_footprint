-- Webpage Report --
To view our webpage report open the index.html file in the Webpage folder.

-- Visualization --
To view our visualization run a local http server in the Visualization folder the open the index.html file.

To run a local server try for example: 
If you have Python 3.x installed run: python -m http.server

Or install Node.js, to install run: npm install http-server -g
Then to start the server run: http-server .
